
## Features

- **Fully Responsive**: Works on mobile, tablet, and desktop
- **Modern Design**: Clean layout matching the provided image
- **Interactive Elements**: 
  - Mobile hamburger menu
  - Smooth scrolling navigation
  - Hover effects on cards and buttons
  - Form validation
- **Performance Optimized**: Minimal and efficient code

## Image Requirements

Place these images in the `/assets/images/` folder:

1. **Hero Image** (`hero.jpg`) - Global team collaboration
2. **Service Icons** (4 files) - 80x80px recommended
3. **About Section Image** (`MK Nexus team collaboration.png`)
4. **Global Map** (`MK Nexus global presence map.png`)
5. **Logo** (`logo.png`) - 200x40px recommended

## Setup Instructions

1. Create the folder structure as shown above
2. Place all code files in the root directory
3. Add required images to `/assets/images/` folder
4. Open `index.html` in a web browser

## Customization

- **Colors**: Modify CSS custom properties in `:root`
- **Content**: Update text in `index.html`
- **Images**: Replace placeholder images with final assets
- **Fonts**: Change `font-family` in CSS
